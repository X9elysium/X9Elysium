Voice settings

Three providers — pick one per voice preset. Browser TTS is free + offline (uses your OS voices). OpenAI and ElevenLabs need an API key (stored locally; never sent anywhere except the chosen provider).

Keys live in localStorage on this machine and are sent only when you hit Play. The proxy at /api/tts forwards them straight to the provider.

Voice presets

Each preset = a voice. For ElevenLabs, paste any voice ID from the voice library — including custom-cloned voices you have rights to use.
